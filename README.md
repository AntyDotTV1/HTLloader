# HTLLoader

**Version:** v1.0.6

HTLLoader is a simple launcher for ArchiSteamFarm (ASF) with a user-friendly console interface and quick links menu.

---

## Features

- Launches ArchiSteamFarm (ASF) from the main folder or plugins.
- Displays application uptime in the console.
- Quick links to Steam Group and GitHub.
- Custom ASCII logo and window title showing version.

---

## Requirements

- Python 3.6 or higher
- ArchiSteamFarm (ASF) installed

---

## Usage

1. Run the `HTLLoader.py` script.
2. Select an option from the menu:
   - **1 – ArchiSteamFarm**: Runs ASF.
   - **2 – Links**: Opens preset links in your web browser.
   - **3 – Exit**: Closes the launcher.
3. The console window title will display `HTLLoader v1.0.6`.

---

## Notes

- The launcher is designed for Windows. The console title will update only on Windows.
- If ASF is not found in the expected directories, an error message will appear.
